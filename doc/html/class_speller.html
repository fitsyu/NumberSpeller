Detailed Description

The Speller class It splits the number into triples, turn each triple into Number object from which the textual form of particular number could be retreived, then form a complete textual form for each triple with their corresponding units respectively.

usage: Create new instance of Speller with number to spell, as in: Speller speller("1123");

The instance will be able to process() only if the number is valid i.e only contains digit character nothing else. the ability can be queried using ready() method.
